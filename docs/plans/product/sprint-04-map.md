# Sprint 4: Карта тренировок и поиск

## Проблема / Краткое описание

Атлеты должны иметь возможность находить тренировки рядом с собой или в выбранном районе. Нужна интерактивная карта с фильтрацией по видам спорта, датам и другим параметрам. Это основной способ discovery тренировок в приложении.

## Ценность для пользователя

**Для атлетов:** Быстрый и удобный поиск подходящих тренировок по местоположению, виду спорта и времени. Визуальное представление помогает выбрать удобное место.

**Для тренеров:** Их тренировки становятся видны целевой аудитории, привлекая новых участников.

**Для бизнеса:** Основная витрина продукта, точка входа в воронку конверсии (просмотр → бронирование → оплата).

## Скоуп спринта

### Включено:
- Публичная карта с маркерами тренировок
- Фильтры: вид спорта, дата, время
- Детальная карточка тренировки (bottom sheet)
- Автоопределение города пользователя
- Выбор города вручную
- Индикация количества свободных мест
- Просмотр без авторизации

### Исключено:
- Фильтр по цене
- Фильтр по рейтингу тренера
- Сохранение тренировок в избранное
- История просмотров
- Рекомендации
- Текстовый поиск
- Маршруты до места тренировки

## Критерии приёмки

### Открытие карты

**Дано:** Любой пользователь (в том числе неавторизованный)
**Когда:** Открывает главную страницу сайта или нажимает "Карта"
**Тогда:**
1. Видит карту с маркерами тренировок
2. Если геолокация разрешена → карта центрируется на пользователе
3. Если геолокация запрещена → показывается выбор города
4. Маркеры показывают только будущие тренировки
5. При большом количестве маркеры группируются в кластеры

### Фильтрация тренировок

**Дано:** Пользователь на карте
**Когда:** Открывает панель фильтров
**Тогда:**
- **Вид спорта:** Чекбоксы (можно выбрать несколько)
- **Дата:** Сегодня / Завтра / Неделя / Выбрать дату
- **Время:** Утро (6-12) / День (12-18) / Вечер (18-24)
- **Только со свободными местами:** Тоггл
- Количество результатов обновляется в реальном времени
- Кнопка "Сбросить фильтры"

### Просмотр карточки тренировки

**Дано:** Пользователь видит маркеры на карте
**Когда:** Кликает на маркер
**Тогда:**
1. Снизу выезжает карточка с информацией:
   - Вид спорта и название (если есть)
   - Фото и имя тренера
   - Дата и время
   - Адрес/описание места
   - Свободно X из Y мест
   - Цена за место
   - Кнопка "Записаться"
2. Можно потянуть карточку вверх для полного просмотра
3. Можно закрыть свайпом вниз или крестиком

### Индикация на маркерах

**Дано:** Маркеры тренировок на карте
**Тогда:**
- Цвет/иконка по виду спорта
- Зелёный индикатор = есть места
- Жёлтый = осталось мало мест (<30%)
- Красный = мест нет
- При наведении показывается краткая информация

### Выбор города

**Дано:** Геолокация не определена или пользователь хочет сменить город
**Когда:** Нажимает на название города в шапке карты
**Тогда:**
1. Открывается список доступных городов
2. Может выбрать город из списка
3. Карта перецентрируется на выбранный город
4. Загружаются тренировки этого города

## UX заметки

### Интерфейс карты:
- Полноэкранная карта с элементами поверх
- Фильтры в виде чипсов сверху
- Счётчик найденных тренировок
- Кнопка "Моё местоположение" для возврата
- Зум-контролы для мобильной версии

### Bottom sheet с деталями:
- **Collapsed (свёрнутое):** Основная информация, кнопка записи
- **Expanded (развёрнутое):** + описание, + информация о тренере
- Smooth animation при открытии/закрытии
- Возможность перейти к профилю тренера (в будущем)

### Состояния загрузки:
- Скелетоны для карточек
- Спиннер при загрузке маркеров
- "Ничего не найдено" при пустых результатах

### Мобильная версия:
- Фильтры в виде bottom drawer
- Жесты для управления картой
- Адаптивные размеры маркеров
- Упрощённая карточка для маленьких экранов

## Валидация и ограничения

### Производительность:
- Максимум 100 маркеров одновременно на экране
- При большем количестве - кластеризация
- Ленивая загрузка при движении карты
- Кеширование результатов на 5 минут

### Фильтры:
- По умолчанию: все виды спорта, ближайшие 7 дней
- Нельзя выбрать прошедшую дату
- Максимальный период - 1 месяц вперёд

## Копирайт и тексты

### Пустая карта:
**Заголовок:** "Нет тренировок в этом районе"
**Текст:** "Попробуйте изменить фильтры или подвигать карту"
**Кнопка:** "Сбросить фильтры"

### Выбор города (первый вход):
**Заголовок:** "Выберите ваш город"
**Текст:** "Мы покажем тренировки рядом с вами"

### Подсказки на карте:
- "Нажмите на маркер для просмотра деталей"
- "Потяните карточку вверх для полной информации"

### Карточка тренировки:
**Свободные места:** "Свободно 5 из 10 мест"
**Нет мест:** "Мест нет"
**Последнее место:** "Осталось 1 место!"

### Фильтры:
**Заголовки секций:**
- "Вид спорта"
- "Когда"
- "Время дня"

## Метрики успеха

- Среднее время до клика на маркер < 30 секунд
- Конверсия просмотр карточки → клик "Записаться" > 20%
- Использование фильтров > 40% сессий
- Bounce rate < 30%
- Среднее количество просмотренных карточек > 3

## Технические требования

### API эндпоинт:
- `GET /api/workouts/map`
- Параметры: bbox (границы видимой области), filters (спорт, дата, время)
- Лёгкий payload: только необходимые поля для маркеров
- Детали подгружаются при клике на маркер

### Оптимизация:
- Дебаунс при движении карты (500ms)
- Кластеризация через leaflet.markercluster
- WebP для изображений тренеров
- Lazy loading для аватаров
