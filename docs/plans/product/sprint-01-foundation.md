# Sprint 1: Регистрация и авторизация

## Проблема / Краткое описание

Необходимо создать базовую систему входа в приложение с разделением пользователей на две роли: атлеты (ищут тренировки) и тренеры (создают тренировки). Каждая роль должна иметь свой личный кабинет с соответствующим функционалом.

## Ценность для пользователя

**Для атлетов:** Возможность создать аккаунт и получить доступ к персональному кабинету для просмотра будущих тренировок.

**Для тренеров:** Возможность зарегистрироваться в системе и получить доступ к инструментам управления тренировками.

**Для бизнеса:** Формирование базы пользователей с чёткой сегментацией по ролям, основа для всех последующих функций.

## Скоуп спринта

### Включено:
- Регистрация с выбором роли (атлет/тренер)
- Вход по email и паролю
- Восстановление пароля
- Базовая навигация для каждой роли
- Разделение доступа по ролям
- Выход из системы

### Исключено:
- Социальная авторизация (OAuth)
- Верификация телефона (SMS)
- Двухфакторная аутентификация
- Переключение между ролями
- Удаление аккаунта

## Критерии приёмки

### Регистрация

**Дано:** Новый пользователь заходит на сайт split.fitness
**Когда:** Нажимает "Регистрация"
**Тогда:**
- Видит форму с полями: Имя, Фамилия, Email, Телефон (опционально), Пароль
- Видит выбор роли: "Я атлет" / "Я тренер"
- После заполнения и отправки формы:
  - Если выбрал "атлет" → попадает в личный кабинет атлета
  - Если выбрал "тренер" → попадает в личный кабинет тренера
- Получает email с подтверждением регистрации

### Авторизация

**Дано:** Зарегистрированный пользователь
**Когда:** Вводит email и пароль на странице входа
**Тогда:**
- При правильных данных попадает в свой кабинет:
  - Атлет → /athlete/bookings
  - Тренер → /coach/dashboard
- При неправильных данных видит сообщение об ошибке

### Разделение доступа

**Дано:** Авторизованный пользователь
**Когда:** Пытается зайти на страницу другой роли
**Тогда:**
- Атлет не может зайти на /coach/* → видит ошибку 403
- Тренер не может зайти на /athlete/* → видит ошибку 403

### Восстановление пароля

**Дано:** Пользователь забыл пароль
**Когда:** Нажимает "Забыли пароль?" и вводит email
**Тогда:**
- Получает письмо со ссылкой для сброса пароля
- По ссылке может установить новый пароль
- После смены пароля автоматически входит в систему

## UX заметки

### Экран регистрации
- Выбор роли должен быть визуально понятным (карточки с иконками)
- Под выбором роли пояснение:
  - "Атлет" - Ищу тренировки и записываюсь на занятия
  - "Тренер" - Провожу тренировки и управляю записями
- Важно: одна роль на аккаунт, без возможности изменения

### Навигация после входа
**Атлет видит в меню:**
- Тренировки (главная)
- Профиль

**Тренер видит в меню:**
- Мои тренировки
- Профиль
- Выплаты (заглушка)

### Мобильная версия
- Все формы должны быть адаптированы для экранов 350-428px
- Автофокус на первом поле формы
- Кнопки достаточно крупные для нажатия пальцем

## Валидация полей

### При регистрации:
- **Email:** Обязателен, формат email, уникальный в системе
- **Пароль:** Минимум 8 символов
- **Имя:** Обязательно, 2-50 символов
- **Фамилия:** Обязательно, 2-50 символов
- **Телефон:** Опционально, если указан - формат +7XXXXXXXXXX
- **Роль:** Обязательна к выбору

### Сообщения об ошибках:
- "Email уже используется" → предложить войти или восстановить пароль
- "Пароль слишком короткий" → показать требования
- Показывать ошибки под соответствующими полями

## Статусы и состояния

### Статус пользователя:
- **active** - может пользоваться системой (по умолчанию)
- **blocked** - заблокирован администратором

### Статус email:
- **unverified** - не подтверждён
- **verified** - подтверждён (после клика по ссылке из письма)

## Копирайт и тексты

### Экран выбора роли:
**Заголовок:** "Кто вы?"

**Карточка атлета:**
- Заголовок: "Я атлет"
- Описание: "Хочу находить и записываться на тренировки"

**Карточка тренера:**
- Заголовок: "Я тренер"
- Описание: "Хочу проводить тренировки и управлять записями"

### Email подтверждения регистрации:
**Тема:** "Добро пожаловать в Split.fitness!"
**Текст:**
```
Здравствуйте, {Имя}!

Спасибо за регистрацию на Split.fitness.

[Подтвердить email]

Если вы не регистрировались на нашем сайте, просто проигнорируйте это письмо.

С уважением,
Команда Split.fitness
```

## Метрики успеха

- Конверсия регистрации > 60% (от начала заполнения до завершения)
- Время на регистрацию < 2 минут
- Процент подтверждённых email > 80%
- Успешных входов с первой попытки > 90%
