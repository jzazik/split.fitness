# Sprint 2: Профили пользователей

## Проблема / Краткое описание

После регистрации пользователи имеют пустые профили. Тренерам нужно представить себя атлетам (фото, опыт, специализация), а атлетам нужна базовая информация для идентификации. Необходим onboarding flow для заполнения профилей после первого входа.

## Ценность для пользователя

**Для атлетов:** Возможность видеть информацию о тренере перед записью на тренировку, что повышает доверие.

**Для тренеров:** Возможность презентовать себя, свой опыт и специализацию для привлечения атлетов.

**Для бизнеса:** Качественные профили тренеров повышают конверсию в бронирования, создают доверие к платформе.

## Скоуп спринта

### Включено:
- Onboarding flow после первой регистрации
- Профиль тренера: фото, био, опыт, виды спорта, город
- Профиль атлета: фото, контактная информация
- Загрузка и обрезка фото профиля
- Редактирование профиля в любой момент
- Индикатор заполненности профиля

### Исключено:
- Верификация дипломов и сертификатов
- Публичная страница профиля тренера
- Рейтинги и отзывы
- Социальные ссылки
- Портфолио работ
- Видео-презентация

## Критерии приёмки

### Onboarding тренера

**Дано:** Тренер только что зарегистрировался
**Когда:** Входит в систему первый раз
**Тогда:**
1. Видит экран приветствия "Давайте создадим ваш профиль"
2. Шаг 1: Загрузка фото (можно пропустить)
3. Шаг 2: Основная информация (имя, фамилия уже заполнены)
4. Шаг 3: О себе (текстовое поле для био, опыт в годах)
5. Шаг 4: Выбор видов спорта (мультиселект из списка)
6. Шаг 5: Выбор города
7. После завершения попадает в личный кабинет
8. Видит уведомление "Ваш профиль отправлен на модерацию"

### Onboarding атлета

**Дано:** Атлет только что зарегистрировался
**Когда:** Входит в систему первый раз
**Тогда:**
1. Видит упрощённый onboarding (2 шага)
2. Шаг 1: Загрузка фото (можно пропустить)
3. Шаг 2: Выбор города (опционально)
4. После завершения попадает на карту тренировок

### Редактирование профиля

**Дано:** Авторизованный пользователь
**Когда:** Заходит в раздел "Профиль"
**Тогда:**
- Видит все поля своего профиля
- Может изменить любое поле
- Изменения сохраняются по кнопке "Сохранить"
- Видит подтверждение об успешном сохранении

### Загрузка фото

**Дано:** Пользователь загружает фото профиля
**Когда:** Выбирает файл изображения
**Тогда:**
- Открывается инструмент обрезки (квадрат)
- После обрезки фото загружается
- Отображается превью
- Можно удалить и загрузить новое

## UX заметки

### Профиль тренера должен содержать:
- **Фото:** Круглый аватар, 200x200px минимум
- **ФИО:** Полностью
- **О себе:** Текст до 500 символов
- **Опыт:** Количество лет (селект от 0 до 30+)
- **Виды спорта:** Чипсы с названиями (минимум 1, максимум 5)
- **Город:** Выпадающий список

### Профиль атлета:
- **Фото:** Опционально
- **ФИО:** Из регистрации
- **Телефон:** Из регистрации
- **Город:** Опционально

### Индикатор заполненности:
- Показывать процент заполнения профиля
- Для тренера 100% = все обязательные поля заполнены
- Для атлета 100% = добавлено фото и город

### Статус модерации тренера:
- **Жёлтая плашка:** "Профиль на модерации"
- **Зелёная плашка:** "Профиль одобрен"
- **Красная плашка:** "Профиль отклонён: [причина]"

## Валидация полей

### Профиль тренера:
- **Фото:** JPG/PNG, максимум 5MB, минимум 200x200px
- **О себе:** 50-500 символов
- **Опыт:** Обязателен к выбору
- **Виды спорта:** Минимум 1, максимум 5
- **Город:** Обязателен к выбору

### Профиль атлета:
- Все поля опциональны, кроме ФИО (уже есть из регистрации)

## Статусы и состояния

### Статус модерации тренера (coach_profiles.moderation_status):
- **pending** - ожидает проверки (по умолчанию)
- **approved** - одобрен, может публиковать тренировки
- **rejected** - отклонён, не может публиковать

### Влияние статуса:
- **pending/rejected:** Может создавать черновики тренировок, но не публиковать
- **approved:** Полный доступ к функционалу

## Копирайт и тексты

### Onboarding тренера:
**Экран 1:** "Добавьте фото"
- Подзаголовок: "Атлеты чаще записываются к тренерам с фото"
- Кнопка: "Загрузить фото" / "Пропустить"

**Экран 2:** "Расскажите о себе"
- Плейсхолдер: "Опишите свой опыт, достижения, подход к тренировкам..."

**Экран 3:** "Ваша специализация"
- Подзаголовок: "Выберите виды спорта, которые вы преподаёте"

**Экран 4:** "Где вы проводите тренировки?"
- Подзаголовок: "Выберите основной город"

### Уведомления о модерации:
**При отправке:** "Ваш профиль отправлен на проверку. Обычно это занимает до 24 часов."

**При одобрении (email):**
```
Поздравляем! Ваш профиль одобрен.
Теперь вы можете публиковать тренировки на карте.
```

**При отклонении (email):**
```
К сожалению, ваш профиль не прошёл проверку.
Причина: [указать причину]
Пожалуйста, внесите изменения и отправьте профиль повторно.
```
